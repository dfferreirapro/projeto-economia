Você é um desenvolvedor front-end especialista em dashboards interativos.

Na mesma pasta do projeto existe um arquivo chamado "base_escolas.json". 
Esse arquivo contém dados de escolas com campos como:

- tipo (TOP10, SOROCABA, VOTORANTIM)
- ranking
- estado, cidade, escola
- rede (Publica/Privada)
- ideb, saeb_lp, saeb_mat
- taxa_aprovacao
- alunos_total, alunos_turma
- professores
- estrutura (biblioteca, lab_ciencias, lab_info, internet, wifi, quadra, refeitorio, acessibilidade)
- tempo_integral, turnos

Sua tarefa é criar um site estilo dashboard moderno, bonito e interativo utilizando apenas HTML, CSS e JavaScript puro (ou Chart.js para gráficos).

### 🎯 Requisitos do Dashboard:

1. Layout moderno (cards + gráficos + tabela)
2. Tela inicial com KPIs:
   - Média IDEB geral
   - Média SAEB (LP e Matemática)
   - Média taxa de aprovação
   - Total de alunos

3. Gráficos:
   - Gráfico de barras: IDEB por escola
   - Gráfico comparativo: SAEB LP vs Matemática
   - Gráfico de pizza: distribuição entre escolas públicas e privadas
   - Gráfico por tipo (TOP10 vs Sorocaba vs Votorantim)

4. Filtros interativos:
   - Por cidade
   - Por tipo (TOP10, SOROCABA, VOTORANTIM)
   - Por rede (Publica/Privada)

5. Tabela dinâmica:
   - Listar todas escolas
   - Ordenar por IDEB, alunos, aprovação
   - Campo de busca por nome da escola

6. Indicadores visuais:
   - Mostrar ícones ou cores para infraestrutura (ex: biblioteca ✔ ou ❌)
   - Destaque para escolas com IDEB alto (>8)

7. Experiência:
   - Responsivo
   - Visual limpo e profissional (estilo Power BI / sistemas SaaS)
   - Animações suaves

8. Estrutura do código:
   - index.html
   - style.css
   - script.js
   - carregar automaticamente o arquivo JSON local

9. Não usar backend. Tudo deve rodar abrindo o index.html.

10. Código completo, pronto para rodar.

### ⚠️ Importante:
- O JSON deve ser carregado dinamicamente (fetch)
- Tratar valores booleanos (0 e 1) como "Sim/Não"
- Garantir que o sistema funcione mesmo com dados grandes

Gere todo o código necessário.
